  • Lesson 5: Flexbox Layout

  • Lesson 7: Responsive Design

    Media queries, breakpoints, mobile-first design.

  • Lesson 8: CSS Transitions & Animations

  • Lesson 9: CSS Positioning

    Relative, absolute, fixed, sticky.

  • Lesson 10: Build a Complete Layout

    Create a full webpage using modern CSS.
